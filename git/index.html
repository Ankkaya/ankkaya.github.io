# git 简明指南

# 基本配置

注册 github 账号, 创建仓库. 本地安装 github 命令行工具 gitbash 下载, 本地环境绑定你的 github 账号信息

$ git config ——global user.name "name"
$ git config ——global user.email "email"
1
2

设置 SSH Key,首先在 gitbash 输入

ls -al ~/.ssh
1

看看是否有 id_rsa.pub 这个文件, 如果没有, 则需要创建新的 SSH 秘钥

$ ssh-keygen -t rsa -C "注册的 github 邮箱"
1

生成完成后, 打开 id_rsa.pub 文件, 复制对应内容到 github Setting->SSH and GPG keys->New SSH Key 中

基本配置完成

# 仓库操作

git init # 本地仓库初始化

git remote add origin git@github.com:Ankkaya/testgit.git # 添加远程项目到本地仓库

git pull # 下载项目到本地

git status # 查看本地文件修改状态

git add . # 添加文件 ("." 为添加所有改动的文件)

git reset # 取消添加文件

git commit -m "msg" # 提交 ("msg" 本次变动内容)

git push # 提交到远程仓库

git remote -v # 查看本地关联远程仓库地址

git remote rm github # 移除远程仓库

# 分支操作

git branch dev # 创建分支 dev
 
git checkout -b dev # 创建并切换到分支 dev

git pull origin <远程分支> # 关联远程分支

git checkout master # 分支切换 master

git branch # 查看所有分支

git branch -D <本地分支> # 删除本地分支
